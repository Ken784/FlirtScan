# Icons 資源目錄

此目錄包含應用程式的自定義圖標 SVG 文件。

## 需要的圖標文件

請從 Figma 設計中導出以下 SVG 圖標並放置在此目錄：

### 對話相關
- `chat_bubble.svg` - 對話泡泡圖標
- `chat_bubble_flipped.svg` - 翻轉的對話泡泡圖標

### 上傳
- `arrow_up_circle.svg` - 上傳圖標（圓圈內向上箭頭）

### 愛心
- `heart.svg` - 填充的愛心圖標
- `heart_outline.svg` - 輪廓愛心圖標

### 導航
- `home.svg` - 首頁圖標（輪廓）
- `home_selected.svg` - 選中的首頁圖標（填充紅色）
- `inbox.svg` - 收件箱圖標（輪廓）
- `inbox_selected.svg` - 選中的收件箱圖標（填充）

### 操作
- `arrow_back.svg` - 返回箭頭
- `arrow_left.svg` - 左箭頭
- `check.svg` - 勾選圖標（綠色）
- `delete.svg` - 刪除圖標
- `camera.svg` - 相機圖標
- `list.svg` - 列表圖標
- `settings.svg` - 設置圖標
- `activity.svg` - 活動圖標

## 導出方式

1. 在 Figma 中選擇圖標組件
2. 右鍵選擇 "Export" 或使用快捷鍵
3. 選擇 SVG 格式
4. 導出到 `assets/icons/` 目錄
5. 確保文件名與上述列表一致
